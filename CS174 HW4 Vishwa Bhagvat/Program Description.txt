This is a web program written in php and html.  It provides some stock information about a company using the Markit on Demand APIs.  The website for this API is http://www.markit.com/product/markit-on-demand and the information is pulled from the json files which are part of Markit On Demand website.  The initial page asks the user for input.  User has to type company name or symbol.  Once the user clicks on search, certain information will be displayed in the form of a table like company name, company symbol, and exchange information.  A "More Info" link will also be provided for the user to click and this will show certain stock information about a particular company.  User also has the option to click on the clear button which will delete all the characters currently present in the text field as well as any tables that currently appear in the webpage.
